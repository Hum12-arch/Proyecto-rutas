# Sistema de Rutas Cortas con Dijkstra

## Descripción

Proyecto desarrollado en Java para calcular la ruta más corta entre diferentes lugares mediante grafos y el algoritmo de Dijkstra.

El sistema permite registrar lugares, registrar conexiones, mostrar el mapa y calcular el recorrido de menor distancia entre un origen y un destino.

## Funcionalidades

- Registrar nuevos lugares.
- Registrar conexiones entre lugares.
- Asignar una distancia a cada conexión.
- Mostrar el mapa de lugares.
- Calcular la ruta más corta.
- Mostrar el recorrido recomendado.
- Mostrar la distancia total.
- Validar lugares y distancias ingresadas.

## Algoritmo utilizado

El sistema utiliza el algoritmo de Dijkstra para calcular la ruta de menor distancia entre un nodo de origen y un nodo de destino.

## Ejemplo

Origen: Casa

Destino: Universidad

Resultado:

Casa → Mercado → Universidad

Distancia total: 6 km
